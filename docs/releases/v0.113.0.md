# v0.113.0

v0.113.0 is a non-breaking review policy release that adds named presets for common worker rollout modes.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Policy presets | `contextdb worker review --policy-preset` | Provides stable rollout defaults for common worker operating modes |
| Config files | JSON `policy_preset` | Lets Norn and other deployment specs use the same preset names as the CLI |
| Rollout modes | Observe-only, stale-only, provider-assisted validation | Makes dry-run and provider-assisted worker adoption easier to configure consistently |

## Verification

- `go test ./...`
- `npm run docs:build`
- Review worker policy preset tests

## Compatibility

This release is additive. Presets apply only when selected; existing explicit policy configuration remains supported.

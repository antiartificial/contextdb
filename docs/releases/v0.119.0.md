# v0.119.0

v0.119.0 is a non-breaking ranking dashboard release that keeps recent local baselines available in the browser.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Ranking dashboard | `/admin/` local baseline history | Stores recent ranking evaluation runs in browser-local history |
| Release comparison | Baseline deltas | Lets operators compare ranking runs without repeatedly reselecting JSON artifacts |
| Review ergonomics | Local-only history | Improves dashboard workflow without adding server-side persistence requirements |

## Verification

- `go test ./...`
- `npm run admin:build`
- `npm run docs:build`
- Admin ranking dashboard baseline coverage

## Compatibility

This release is additive. Browser-local baseline history does not change ranking scoring, stored data, REST APIs, GraphQL APIs, or SDK behavior.

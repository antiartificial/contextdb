# v0.120.0

v0.120.0 is a non-breaking Postgres reliability release that adds negative-path integration coverage.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Fault injection | Docker-backed Postgres integration tests | Covers malformed DSNs, unreachable Postgres, and read-only migration failures |
| Store readiness | Postgres diagnostics | Makes failed Postgres setup easier to detect before production writes |
| Integration coverage | Write/retrieve smoke | Keeps normal Postgres storage behavior covered beside the new failure cases |

## Verification

- `go test ./...`
- `go test -tags integration ./pkg/client`
- `npm run docs:build`

## Compatibility

This release is additive. Postgres behavior remains compatible; the release adds integration coverage and clearer readiness evidence.

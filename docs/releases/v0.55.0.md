# v0.55.0 Release Recap

v0.55.0 is a non-breaking release-review slice that adds Markdown recaps for ranking evaluations.

## What Changed

| Area | New in v0.55.0 | Why it matters |
|:-----|:---------------|:---------------|
| Ranking recap | `contextdb eval ranking --markdown` | Prints a compact human-readable ranking summary |
| Recap file output | `contextdb eval ranking --markdown-out PATH` | Lets release reviewers attach MRR, failures, top results, and score breakdowns to release notes |
| JSON compatibility | Existing `--out` and `--report` behavior | Keeps machine-readable snapshot workflows unchanged |
| Feature metadata | `ranking-eval-markdown-recap` | Lets live deployments advertise Markdown ranking recap support |

## Compatibility

This release only adds optional CLI output modes. Existing JSON snapshots, ranking behavior, REST APIs, storage formats, and SDK contracts are unchanged.

## Operator Flow

```bash
contextdb eval ranking --out ranking-eval.json --markdown-out ranking-eval.md
```

Use `--markdown` for stdout output during local release review. Use `--markdown-out` when saving an artifact beside the JSON snapshot.

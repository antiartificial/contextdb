# v0.93.0 Release Recap

v0.93.0 is a non-breaking debugger release that adds side-by-side explain-rank comparison to the Svelte admin dashboard.

## What's New

| Area | New in v0.93.0 | Why it matters |
|:-----|:---------------|:---------------|
| Admin compare API | `/admin/api/explain-rank` | Lets the observe-port debugger call the existing explain-rank engine without switching to the public REST port |
| Svelte compare panel | Search results can fill left/right compare targets | Moves operators from finding candidate nodes to comparing rank factors in one workflow |
| Rank evidence output | Summary plus raw explain-rank JSON | Preserves full factor deltas, winner/loser IDs, scores, and evidence details for incident notes |
| Feature metadata | `admin-debugger-explain-rank` | Lets live deployments advertise dashboard rank comparison support |

## Operator Flow

Open the dashboard and search for candidates:

```bash
open http://localhost:7702/admin/
```

Call the compare API directly:

```bash
curl -X POST http://localhost:7702/admin/api/explain-rank \
  -H 'content-type: application/json' \
  -d '{"namespace":"my-app","node_id":"NODE_UUID","other_node_id":"OTHER_NODE_UUID","text":"optional query"}'
```

## Compatibility

This release is intentionally non-breaking. Existing admin routes, metrics, search, belief audits, REST, GraphQL, SDK, and manifest verification behavior are unchanged.

# v0.115.0

v0.115.0 is a non-breaking admin operations release that turns review worker posture into a dashboard surface.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Review ops API | `/admin/api/review-ops` | Surfaces review queue size, stale claims, source trust drops, worker action counts, recent runs, and manual-review lanes |
| Admin UI | `/admin/` review panels | Lets operators inspect review health without stitching together logs and API calls |
| Worker observability | Recent run summaries and action counts | Connects review worker activity to current queue and source trust posture |

## Verification

- `go test ./...`
- `npm run admin:build`
- `npm run docs:build`
- Admin review operations API and embedded asset coverage

## Compatibility

This release is additive. Existing admin APIs, review queues, worker execution, and SDK calls remain compatible.

# v0.11.0 Release Recap

v0.11.0 is a non-breaking inspectability release that adds graph evidence to explain-rank responses. Rank explanations now include supporting evidence chains and compound confidence for each compared node.

## What Changed

| Area | New in v0.11.0 | Why it matters |
|:-----|:---------------|:---------------|
| Go SDK evidence | `RankedNodeExplanation.Evidence` | Lets applications inspect supporting graph paths behind each ranked node |
| REST evidence | `node.evidence` and `other.evidence` | Gives operators support counts, links, edge weights, and compound confidence |
| GraphQL evidence | `evidence { supportCount links { ... } }` | Gives debugger UIs graph-friendly rank evidence |
| Feature metadata | `explain-rank-graph-evidence` feature tag | Lets live deployments advertise evidence-enriched rank explanations |

## Non-Breaking Behavior

The evidence fields are additive. Existing explain-rank requests and score deltas continue to work unchanged.

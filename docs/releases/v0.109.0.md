# v0.109.0

v0.109.0 is a non-breaking review operations release that makes claim review worker cycles auditable across server and SDK surfaces.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Worker audit | Durable `review_worker_run` events | Records cycle id, evaluator mode, dry-run flag, counts, and planned decisions for each worker pass |
| API access | Go SDK, REST, GraphQL, TypeScript, and Python | Lets operators and agents inspect run summaries without scraping logs |
| Release readiness | Dry-run review evidence | Gives review-worker rollouts a durable evidence trail before any mutation is enabled |

## Verification

- `go test ./...`
- `npm run docs:build`
- SDK contract coverage for review worker run summaries

## Compatibility

This release is additive. Review worker run summaries add new event and API fields without changing existing write, retrieve, ranking, feedback, or review queue behavior.

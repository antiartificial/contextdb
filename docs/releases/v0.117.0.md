# v0.117.0

v0.117.0 is a non-breaking source trust release that makes quarantine candidates visible and reviewable from the admin surface.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Quarantine candidates | `/admin/api/review-ops` | Includes dry-run source quarantine candidates beside review worker posture |
| Reviewed execution | `/admin/api/source-quarantine` | Applies reviewed source labels one source at a time |
| Operator safety | Dry-run-first source trust workflow | Keeps low-trust source handling visible before source labels change |

## Verification

- `go test ./...`
- `npm run admin:build`
- `npm run docs:build`
- Source quarantine admin API coverage

## Compatibility

This release is additive. Source labels change only through explicit reviewed execution; existing source trust, retrieval, and review behavior remains compatible.

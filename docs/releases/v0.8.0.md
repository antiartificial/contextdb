# v0.8.0 Release Recap

v0.8.0 is a non-breaking release that adds explain-rank APIs. Applications can compare two nodes under the current scoring model and see which score components created the ranking difference.

## What Changed

| Area | New in v0.8.0 | Why it matters |
|:-----|:--------------|:---------------|
| Go SDK explain-rank | `NamespaceHandle.ExplainRank(ctx, req)` | Lets applications inspect why one candidate outranks another |
| REST explain-rank | `POST /v1/namespaces/{ns}/rank/explain` | Gives operators and tests a simple rank debugging endpoint |
| GraphQL explain-rank | `explainRank(namespace:, nodeId:, otherNodeId:)` | Gives debugger UIs a graph-friendly score comparison |
| Feature metadata | `explain-rank` feature tag | Lets live deployments advertise rank explanation support through introspection |

## Non-Breaking Behavior

Explain-rank is additive. It does not change retrieval ranking, score weights, or write behavior; it exposes the scoring factors already used by retrieval.

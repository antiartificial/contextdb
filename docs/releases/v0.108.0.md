# v0.108.0

v0.108.0 is a non-breaking reliability release. It turns several follow-up items into executable checks: CI-backed release health artifacts, schema catalog drift annotations, closure bundle re-verification, a fixture catalog, dry-run-first source quarantine, and harder durability lanes.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Release health | CI `release-health` artifact | Generates JSON and Markdown recap rows from actual test, build, durability, Postgres, and Docker job results |
| Durability | CI `durability` and `postgres integration` jobs | Adds race-sensitive package coverage, a concurrent write/retrieve soak test, and Docker-backed Postgres smoke coverage |
| Schema catalog | `contextdb docs schema-catalog verify --annotations` | Emits compact CI annotations when public schema catalog entries point to missing or mismatched artifacts |
| Closure bundles | `contextdb snapshot lifecycle index publish closure-bundle verify --manifest PATH` | Re-hashes saved published-backup repair evidence bundles against their manifest |
| Fixture catalog | `/fixtures/index.json` | Makes the KV refresh receipt fixtures discoverable for integration and CI consumers |
| Source quarantine | `NamespaceHandle.SourceQuarantine` | Builds dry-run plans for repeatedly refuted or low-trust sources and only applies quarantine labels when `Execute` is set |

## Verification

- `go test -count=1 ./cmd/contextdb ./testdata`
- `go test -count=1 ./pkg/client`
- `go test -race -count=1 ./internal/federation ./internal/retrieval ./pkg/client`
- `go test -tags integration -count=1 ./pkg/client -run TestPostgresStandardModeWriteRetrieveSmoke` (skips locally unless `CONTEXTDB_TEST_POSTGRES_DSN` is set)

## Compatibility

This release is intentionally non-breaking. The new CLI commands, SDK method, CI jobs, docs artifacts, and feature metadata are additive. Existing APIs, storage formats, ranking behavior, and default write paths remain unchanged unless an operator explicitly executes a source quarantine plan.

# v0.86.0 Release Recap

v0.86.0 is a non-breaking documentation release that adds a GitHub Actions workflow recipe for ranking baseline manifest verification artifacts and annotations.

## What's New

| Area | New in v0.86.0 | Why it matters |
|:-----|:---------------|:---------------|
| CI recipe | Ranking baseline verification GitHub Actions snippet | Shows how to keep JSON, Markdown, and annotation artifacts from one verifier run |
| Annotation replay | `cat ranking-baseline-manifest-annotations.txt` | Lets CI surface saved annotation lines while preserving the verifier exit status |
| Artifact upload | `actions/upload-artifact@v4` path list | Keeps release evidence available after pass or fail |
| Feature metadata | `ranking-baseline-annotation-fixture-docs` | Lets deployments advertise the documented CI annotation workflow |

## Operator Flow

```yaml
- name: Verify ranking baseline manifest
  run: |
    set +e
    contextdb eval ranking baseline manifest verify \
      --manifest ranking-baseline-manifest.json \
      --report \
      --markdown-out ranking-baseline-manifest-verification.md \
      --annotations-out ranking-baseline-manifest-annotations.txt
    status=$?
    cat ranking-baseline-manifest-annotations.txt
    exit "$status"
```

The full recipe also uploads the manifest, Markdown recap, and annotation output as release artifacts.

This release is intentionally non-breaking. It changes documentation and feature metadata only.

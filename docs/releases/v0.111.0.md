# v0.111.0

v0.111.0 is a non-breaking worker reliability release that adds evaluator smoke tests before claim review opens the database.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Evaluator smoke | `contextdb worker review --smoke-evaluator` | Runs synthetic non-mutating checks for rules, webhook, provider, and hybrid evaluator configuration |
| Startup safety | Pre-database validation | Catches broken evaluator configuration before a worker can inspect queues or plan decisions |
| Deployment evidence | Smoke report output | Gives Norn and local operators a quick pass/fail signal for review worker readiness |

## Verification

- `go test ./...`
- `npm run docs:build`
- Review worker smoke evaluator tests

## Compatibility

This release is additive. Smoke checks run only when requested and do not mutate claims, sources, queues, or review decisions.

# v0.84.0 Release Recap

v0.84.0 is a non-breaking ranking release that adds Markdown summaries for ranking baseline manifest verification.

## What's New

| Area | New in v0.84.0 | Why it matters |
|:-----|:---------------|:---------------|
| Markdown summary | `contextdb eval ranking baseline manifest verify --markdown` | Prints a compact release-review recap for manifest verification |
| Summary artifact | `--markdown-out PATH` | Saves the verification recap beside JSON reports in CI |
| Artifact table | Version, kind, retention status, result, and bytes | Makes retained, verified, and expected-missing artifacts quick to scan |
| Feature metadata | `ranking-baseline-manifest-verify-markdown` | Lets live deployments advertise manifest verification Markdown support |

## Operator Flow

```bash
contextdb eval ranking baseline manifest verify \
  --manifest ranking-baseline-manifest.json \
  --report \
  --markdown-out ranking-baseline-manifest-verification.md
```

The Markdown recap includes pass/fail status, manifest metadata, artifact totals, validation errors, and a compact artifact summary table.

## Compatibility

This release is intentionally non-breaking. Existing JSON verification reports and exit behavior are unchanged unless `--markdown` or `--markdown-out` is supplied.

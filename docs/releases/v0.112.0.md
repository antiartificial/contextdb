# v0.112.0

v0.112.0 is a non-breaking SDK parity release for async Python agents that participate in review workflows.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Async Python | `AsyncNamespace` review APIs | Exposes review queues, review decision recording, decision listing, and worker run summaries |
| Agent workflows | Async review participation | Lets async agents inspect review posture and leave review decisions without switching client modes |
| SDK consistency | Sync and async parity | Keeps Python review workflow behavior aligned across both client styles |

## Verification

- `go test ./...`
- `npm run docs:build`
- Python SDK async review workflow tests

## Compatibility

This release is additive. Existing Python sync clients and non-review SDK methods continue to behave unchanged.

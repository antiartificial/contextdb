# v0.89.0 Release Recap

v0.89.0 is a non-breaking debugger release that adds graph-backed search to the admin belief debugger.

## What's New

| Area | New in v0.89.0 | Why it matters |
|:-----|:---------------|:---------------|
| Debugger search API | `/admin/api/search?ns=NS&q=QUERY&limit=N` | Finds recent valid nodes by text, source, label, or node ID without requiring embeddings |
| Debugger search UI | Search panel on `/admin/` and `/admin/debugger` | Lets operators find candidate nodes and open them in the belief debugger |
| Embedded-friendly search | Graph-store scan over valid nodes | Works in local and Mini/Norn deployments even when no embedder is configured |
| Feature metadata | `admin-debugger-search` | Lets live deployments advertise debugger search support |

## Operator Flow

Open the dashboard:

```bash
open http://localhost:7702/admin/
```

Search recent valid nodes:

```bash
curl "http://localhost:7702/admin/api/search?ns=my-app&q=auth&limit=10"
```

Open one returned node in the debugger:

```bash
curl "http://localhost:7702/admin/api/belief?ns=my-app&id=NODE_UUID"
```

## Compatibility

This release is intentionally non-breaking. Existing admin stats, belief debugger, REST, GraphQL, SDK, and manifest verification behavior are unchanged.

# v0.118.0

v0.118.0 is a non-breaking debugger release that adds relation and source trust filtering to graph context.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Belief debugger | `/admin/api/belief` | Filters graph neighbors by relation and source trust band |
| Admin UI | `/admin/` graph context controls | Lets operators narrow support, contradiction, provenance, and trust views while debugging a claim |
| Source credibility | Neighbor source trust metadata | Makes surrounding graph evidence easier to judge during review |

## Verification

- `go test ./...`
- `npm run admin:build`
- `npm run docs:build`
- Admin belief debugger filter coverage

## Compatibility

This release is additive. Existing belief debugger responses keep their prior fields, with filter inputs and source credibility metadata added for callers that use them.

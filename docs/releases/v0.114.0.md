# v0.114.0

v0.114.0 is a non-breaking evaluator reliability release that hardens smoke tests around provider failure modes.

## What's New

| Area | Surface | Why it matters |
|:-----|:--------|:---------------|
| Failure fixtures | Evaluator smoke tests | Covers missing provider keys, rate limits, and malformed provider decision responses |
| Provider readiness | Smoke failure reporting | Confirms provider-backed evaluators fail clearly without mutating claims |
| Hybrid evaluation | Provider and fallback checks | Helps operators validate hybrid review-worker setups before deployment |

## Verification

- `go test ./...`
- `npm run docs:build`
- Evaluator smoke failure fixture tests

## Compatibility

This release is additive. It expands test coverage and smoke reporting without changing default rules evaluator behavior.

# v0.56.0 Release Recap

v0.56.0 is a non-breaking ranking review release that compares ranking eval snapshots across releases.

## What Changed

| Area | New in v0.56.0 | Why it matters |
|:-----|:---------------|:---------------|
| Snapshot comparison | `contextdb eval ranking --compare previous.json` | Compares the current representative corpus snapshot against a saved baseline |
| JSON diff output | `--diff-report` and `--diff-out PATH` | Captures MRR, pass-count, rank, and top-score movements for automation |
| Markdown diff output | `--diff-markdown` and `--diff-markdown-out PATH` | Produces a release-review summary of the largest ranking movements |
| Feature metadata | `ranking-eval-historical-diff` | Lets live deployments advertise ranking snapshot comparison support |

## Compatibility

This release only adds optional CLI comparison output. Existing ranking behavior, JSON snapshots, Markdown recaps, REST APIs, storage formats, and SDK contracts are unchanged.

## Operator Flow

```bash
contextdb eval ranking \
  --compare previous-ranking-eval.json \
  --diff-markdown-out ranking-diff.md \
  --out current-ranking-eval.json
```

Use the JSON diff for automation and the Markdown diff for release review notes.
